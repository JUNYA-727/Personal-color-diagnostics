# Description
深層学習を用いて手軽にパーソナルカラー診断を行えることで今後の服選ぶや女性の方なら化粧品選びの参考になれば良いと思ったから作成に至った｡
 
# Features
パーソナルカラー診断を行いたい画像のパスを与えることで､その画像に対して春､夏､秋､冬の各タイプの可能性を提示する｡
また､最も近いタイプを表示する｡
 
# Requirement
 
detect.pyを使用するためのライブラリ
 
* tensorflow 2.5.0
* opencv 4.5.2.52
* dlib 19.22.1
* imutils 0.5.4
 
# Installation
 
Requirementで列挙したライブラリなどのインストール方法を説明する
